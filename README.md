# Willis Baptist Redesign

Standalone Next 16 redesign concept for Willis Baptist Church.

- Local folder: `willisbaptist-redesign`
- Production URL: `https://willisbaptist-redesign.vercel.app`
- Original site reference: `https://www.willisbaptist.org/`

## Commands

```bash
npm install
npm run dev
npm run build
```

Note: on this machine, `npm` was run successfully through Rosetta using `arch -x86_64 /usr/local/bin/npm ...` because the default `node` binary is x86_64-only.
